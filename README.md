# LLM Word Game Arena
 
### Prerequisites
1. Python
2. Ollama

### Instructions
1. Clone the repository
2. Install requirements
```bash
pip install -r requirements.txt
```
3. Download the models
```bash
ollama pull llama3:8b-instruct-q8_0
ollama pull gemma2:9b-instruct-q8_0 
```
4. Run the script
```bash
python main.py
```
